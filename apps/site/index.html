Self-host in your account

From clone to first link.

The guided bootstrap previews every resource name, checks the selected account and requires an exact confirmation before creating anything.

  1. 01

    Clone and install

    Fork the GPL-licensed repository and install the pinned workspace dependencies.

    npm install
  2. 02

    Preview resources

    Generate unique D1, KV, Worker and Pages names without changing Cloudflare.

    npm run deploy:bootstrap
  3. 03

    Approve and deploy

    Bind the reviewed release, commit and migrations, then let the safety-gated workflow deploy.

    git push origin main
Fresh installs and upgrades are different tracks.

Existing deployments keep their bindings and require a verified backup before migrations or deploy.
